# homebrew-tap

Official Homebrew tap for Vibooks desktop and CLI releases.

Install:

```bash
brew install --cask vibooks-ai/tap/vibooks
brew install vibooks-ai/tap/vibooks-cli
```

Repository layout:

- `Casks/` contains the generated Vibooks desktop cask
- `Formula/` contains generated formulae such as `vibooks-cli`

Operational notes:

- keep this repository public so `brew install` works without private GitHub access
- `Casks/vibooks.rb` is generated by the `vibooks-app` production release workflow
- `Formula/vibooks-cli.rb` is generated by the `vibooks-cli` production release workflow
- do not hand-edit generated release files unless release automation is unavailable
